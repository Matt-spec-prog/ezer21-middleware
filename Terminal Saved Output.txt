❯ what are all of these things for? github obviously, what else?                                                                  

⏺ Good question. Here's why we need each one:

  Node.js — this is the engine that runs your middleware service. Like how Excel needs to be installed to open a spreadsheet,
  Node.js needs to be installed to run JavaScript code outside of a browser.

  Git — tracks every change you make to your code, like a detailed "undo history." You already have this one.

  Homebrew — a package manager for Mac. Think of it as an App Store for developer tools — it's the easiest way to install things
  like the GitHub CLI. Once it's installed you rarely think about it again.

  GitHub CLI (gh) — lets you create and manage your GitHub repositories directly from Terminal without having to click around on
  the GitHub website. We use it to create your repo and push your code up in one step.
